Funcional:

1- El sistema debe ser capaz de recibir un Pedido de Recolección, teniendo una interfaz para cliente, recolector y jefe de zona. 

2- El sistema debe permitir al usuario entregar sus datos.

3- El sistema debe permitir generar una ruta eficiente para recoger los materiales según los datos del usuario.

4- El sistema debe permitir finalizar la recolección con un comprobante para el usuario.

5- El sistema debe permitir el acceso a notificaciones.

6- El sistema debe permitir al usuario saber la llegada de los recolectores.

7- El sistema debe permitir coordinar la entrega del material.

8- El sistema debe validar la ubicación.




No funcional:

1- El sistema funciona solo con una red de internet estable.

2- El sistema debe garantizar la seguridad de los datos de los clientes.
